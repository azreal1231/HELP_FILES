## Start Nginx
```
service nginx start
```

## Stop Nginx
```
service nginx stop
```
*   OR
```
killall -9 nginx
```

## Quit Nginx
```
service nginx quit
```

## Restart Nginx
```
service nginx restart
```

## Reload Nginx
```
service nginx reload
```

## View Server Status
```
service nginx status
```

## Test Nginx Configuration
```
nginx -t
```

## Check Nginx Version
```
service nginx -v
```

## Show Command Help
```
service nginx -h
```
